## How to run?
### 1. Clone the Repository
```bash 
git clone https://github.com/zaxon-b/7403-microserver-voting-app.git
```

### 2. Build the containers
```bash
cd 7403-microservice-voting-app
```
```bash
docker compose up --build
```

### 3. Vote and check results
Voting Page → http://localhost:6110  
Result Page → http://localhost:6111

### 4. Stop the containers
```bash
docker compose down
```
